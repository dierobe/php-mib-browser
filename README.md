# php-mib-browser

A simple MIB browser with built PHP

<h3>Linux Installation</h3>

Run command: <br>apt-get install php php-snmp
<br>
Or enable SNMP on your PHP installation

<h3>Usage</h3>

Place the project in your web server directory and enjoy!
